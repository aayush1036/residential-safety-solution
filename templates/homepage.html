Info

This project aims at solving the problem of unauthorized vehicles within the society permises which may cause inconvinience to the members of the society. People can register themselves and their vehicles by using the buttons given below to authorize themselves. This object detection model used in this project is trained on the Car license plate detection dataset and the model used for detecting the license plates is the Ultralytics YOLO V5 model and the architecture used in this project is the yolv5s architecture which is lightweight and has a high speed in detecting the license plates in the image. The source code used to train the model is published onKaggle so please upvote the kernel if you like the project. I would really appreciate it.
